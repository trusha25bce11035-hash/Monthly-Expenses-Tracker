Problem Statement

Individuals struggle to track and analyze monthly expenses effectively
Lack of insights into spending patterns and future expense forecasting
Manual expense tracking is time-consuming and error-prone
Scope of the Project

Desktop-based monthly expense management system
Core CRUD operations for expense tracking
Basic ML predictions for future expenses
Text-based data visualization and analytics
Secure user authentication and data persistence
Target Users

Individuals managing personal finances
Students tracking monthly budgets
Freelancers monitoring business expenses
Families managing household spending
High-Level Features

Secure user authentication with password protection
Add, view, update, and delete expenses
Categorize expenses (Food, Transport, Utilities, etc.)
Predict future expenses using moving average algorithm
Visualize spending patterns with text-based charts
Generate expense statistics and monthly summaries
Data persistence with JSON file storage
